# @a-ui/react

A-UI React Components using Core Branding 6.5.0.

## Getting Started

- `npm install @a-ui/react`

- Import the core branding to style the components:
  `@import url("https://cdn.antwerpen.be/core_branding_scss/6.5.0/main.min.css");`
